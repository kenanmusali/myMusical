Adagio for Strings

1936

Description

Adagio for Strings creates an uneasy, shifting suspension as the melody begins a stepwise motion, like the hesitant climbing of stairs.

Chord

features a poignant arrangement for first violins, second violins, violas, cellos, and double basses.
